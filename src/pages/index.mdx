# About Me

***Readable for humans. Precise for machines.***

I'm a technical writer and documentation leader specializing in developer-focused content, REST APIs, and identity technology. I build structured, scalable docs-as-code systems that support both rapid iteration and long-term maintainability. With 13 years of engineering experience, I bring a deep technical foundation to every project.

I collaborate closely with product, engineering, and customer teams to create documentation that is precise, observable, and aligned with product goals. I'm passionate about building documentation that empowers developers, reduces support load, and accelerates adoption.

I also write docs for a second audience: AI coding agents. Applying the AI Agent Spec and other AI-ready documentation standards, I've helped shape content that LLMs can parse and act on reliably, contributing to a 98/100 AI Agent Score on our developer site.

I'm currently exploring opportunities in developer experience, API documentation, technical writing leadership, and content strategy.
